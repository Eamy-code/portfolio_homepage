css detail

今回特に拘ったポイントは side navigationbarのcssの当て方です。

asideをsidebarに設定したかったためhtmlのcodeにあるように containerタグをつけdisplay:flex;をあてました。 さらにheightを100vhにすることで固定した際に余白が出ないよう工夫しました。 また画面全体を15:85の比率にして表示したかったのでflex:1 1 15%;をあてています。

それぞれのaタグのlinkはhoverをつけカーソルを合わせると反応するようにしています。 また、ulタグのみ差別化し中央に配置することで直感的にnavigatio機能だとわかるように工夫しました。

Eamy-codeはspanタグを使用して色を印象付けるために変えたのですが backgroundcolorを統一して1色にしてしまうとどちらかの色が見にくくなるためグラデーションをつけて Eamy-とcodeどちらの色も見やすいよう工夫しました。
