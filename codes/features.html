Site features

responsive design

当ウェブサイトは、最新のウェブデザイントレンドを取り入れ、 訪問者がどのデバイスからでも快適に閲覧できるように心がけています。 スマートフォン、タブレット、デスクトップなど、 様々な画面サイズに対応しており、 視覚的な一貫性を保ちながら最適な表示ができます。

header design

当サイトのヘッダーは、JavaScriptを使用しており、 視覚的な印象を強くしています。 クリックすることで、ユーザーは幾何学的でおもしろい体験ができます。 サイト訪問者に新しい視覚的な印象を提供しています。

navigation system

当サイトにaタグを用いることでナビゲーションシステムを使用してます。 ヘッダーの▼を押すことでbody部分へのナビゲーション、 また右下の▲でホームページの最上部へナビゲーションができるよう設定しました。

information

サイト内の色彩や配置には、 色相環を参考にして同系色を選ぶことで一貫性が生まれています。 これにより、訪問者は直感的に情報を把握しやすくなります。 同系色の統一されたデザインは、視覚的な認識のしやすさを追求し、 ウェブサイト全体に高い統一感をもたらしています。

side bar

サイドバーは、ウェブサイトのナビゲーションバーにもなります。 width（幅）の調整が行い、右側に余分な空白がなくしました。 さらに、スクロール追従機能も追加しました。 これによりページ内のどこにいてもリンクから指定の場所へ飛べます。
